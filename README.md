# HELPER-WORD

## Описание
Веб-приложение для автоматического форматирования текста курсовой работы в формат Word (.docx) с учетом требований ГОСТ.

## Функциональность
- Загрузка текста из файлов (.txt, .doc, .docx) или вставка текста вручную
- Настройка параметров форматирования:
  - Шрифт (Times New Roman, Arial, Courier New)
  - Размер шрифта (12, 14, 16 pt)
  - Междустрочный интервал (1.0, 1.5, 2.0)
  - Отступ красной строки (1.25, 1.5, 2.0 см)
  - Отступы страницы
- Предварительный просмотр текста
- Экспорт в формат .docx с сохранением всех настроек

## Запуск приложения
1. Установите Python (если еще не установлен)
2. Запустите сервер:
   ```bash
   python3 server.py
   ```
3. Откройте в браузере: http://localhost:8000/index.html

## Использование
1. Загрузите текстовый файл или вставьте текст в поле ввода
2. Настройте параметры форматирования по требованиям
3. Нажмите "Предварительный просмотр" для проверки
4. Нажмите "Форматировать и скачать DOCX" для получения готового документа
