# Форматирование текста курсовой работы

Это веб-приложение позволяет загружать текст курсовой работы и автоматически форматировать его в соответствии с академическими требованиями, а затем экспортировать в формате .docx.

## Особенности

- Загрузка текста из файлов (.txt, .docx) или вставка текста вручную
- Настройка параметров форматирования:
  - Шрифт (Times New Roman, Arial, и др.)
  - Размер шрифта
  - Междустрочный интервал
  - Размер красной строки
  - Отступы (верх, низ, лево, право)
  - Отступ между абзацами
- Предварительный просмотр отформатированного текста
- Экспорт в формат .docx с сохранением всех настроек

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Библиотеки:
  - [docx](https://github.com/dolanmiu/docx) - для создания документов Word
  - [mammoth](https://github.com/mwilliamson/mammoth.js) - для чтения DOCX файлов

## Использование

1. Откройте приложение в браузере
2. Загрузите текст курсовой работы:
   - Перетащите файл в область загрузки
   - Или нажмите "Выбрать файл" для загрузки
   - Или вставьте текст непосредственно в текстовое поле
3. Настройте параметры форматирования по вашим требованиям
4. Нажмите "Применить форматирование" для предварительного просмотра
5. Нажмите "Экспорт в DOCX" для скачивания отформатированного документа

## Запуск приложения

Приложение запускается с помощью встроенного HTTP-сервера Python:

```bash
cd /workspace
python3 server.py
```

Затем откройте в браузере: http://localhost:8000

## Форматирование

Приложение автоматически:
- Применяет красную строку (абзацный отступ)
- Устанавливает нужный шрифт и размер
- Настраивает междустрочный интервал
- Устанавливает поля документа
- Применяет выравнивание по ширине
- Обрабатывает заголовки и абзацы по-разному
